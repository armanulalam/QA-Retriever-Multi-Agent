# Get Started
1. Clone the repository
    ``` bash
    git clone https://github.com/armanulalam/QA-Retriever-Multi-Agent.git
    ```
2.Create a virtual environment
    ``` bash
    python -m venv crew
    ```
3.Install the requirements
    ``` bash
    pip install -r requirements.txt
    ```
4.Run the `main.py` file
    ``` bash
    python main.py
    ```
